## 2. Вариант A: Base64 (удобно для Railway)

В Railway не нужно вставлять многострочный JSON — можно одну строку Base64.

**На компьютере** в папке проекта выполни:

```bash
cd d:\appka
python -c "import base64, json; f=open('debet-485119-31d092561d4c.json','rb'); print(base64.b64encode(f.read()).decode())"
```

Скопируй **всю** выведенную строку (без пробелов и переносов).

В Railway → Variables → **New Variable**:
- **Name:** `GOOGLE_CREDENTIALS_BASE64`
- **Value:** вставь скопированную строку.

Сохрани. Railway перезапустит сервис.

---

## 2. Вариант B: обычный JSON

- **Name:** `GOOGLE_CREDENTIALS_JSON`
- **Value:** целиком содержимое файла `debet-485119-31d092561d4c.json` (от `{` до `}`).

Если при вставке что-то обрезается или ломается — используй вариант A (Base64).
